Link da documentação: https://akveo.github.io/nebular/docs/guides/install-nebular#manually

Comando para rodar criação do projeto:

* ng new to-do-list-project-with-nebular

Comando para criar componente:

* ng g component components/todolist

criar o arquivo module para uso em todo o programa, o nome é app.module.ts:

Sobre os comandos a cima:

ng - Todo comando Angular inicia-se com esse prefixo;
g - Seria abreviação para generate;
components/ - vai identificar essa pasta, caso não exista será criada;
todo - Nome do component

Instalar o Nebular para fazer as operações:

* ng add @nebular/theme

* Caso o comando de cima não funcione bem é bom verificar se os comandos de baixo funcionaram devidamente. Para configurar o Nebular é necessário ir no app.module e escrever os comandos a baixo:

import { NbThemeModule } from '@nebular/theme';

...

@NgModule({
  imports: [
    ...
    // this will enable the default theme, you can change this by passing `{ name: 'dark' }` to enable the dark theme
    NbThemeModule.forRoot(),
  ]
})
export class AppModule {



* ir no arquivo angular.json e escrever as funcionalidades a baixo para adiciona-las. Ele está fora do src:

"styles": [
  "node_modules/@nebular/theme/styles/prebuilt/default.css", // or dark.css
],

Pronto! O Nebular estará instalado
