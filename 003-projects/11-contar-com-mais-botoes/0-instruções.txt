Comando para rodar criação do projeto: 

* ng new contar-com-mais-botões

Comando para criar componentes:

* ng g component components/contador

Comando para iniciar a aplicação:

* ng serve

Sobre os comandos a cima: 

ng - Todo comando Angular inicia-se com esse prefixo;
g - Seria abreviação para generate;
components/ - vai identificar essa pasta, caso não exista será criada;
todo - Nome do component

Organização dos componentes na pasta:

src/

│  ├─ components/
│  │  ├─  contador/
│  │  │	├─ contador.component.css

│  │  │    ├─ contador.component.spec.ts
│  │  │    └─ contador.component.ts
│  ├─ app.component.css
│  ├─ app.component.html
│  ├─ app.component.spec.ts
│  ├─ app.component.ts
│  ├─ app.config.ts
│  ├─ app.routes.ts
├─ index.html
├─ main.ts
└─ styles.css
