Comando para rodar criação do projeto:

* ng new todolistprojeto

Comando para criar componente:

* ng g component components/todolist

criar o arquivo module para uso em todo o programa, o nome é app.module.ts:

Sobre os comandos a cima:

ng - Todo comando Angular inicia-se com esse prefixo;
g - Seria abreviação para generate;
components/ - vai identificar essa pasta, caso não exista será criada;
todo - Nome do component

Instalar o boodstrap para fazer as operações:

* npm install bootstrap@5.3.8

ir no arquivo angular.json e escrever as funcionalidades a baixo para adiciona-las. Ele está fora do src:

"styles": [
  "node_modules/bootstrap/dist/css/bootstrap.min.css",
  "src/styles.css"
],
"scripts": [
  "node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"
]

Pronto! O boodstrap estará instalado
